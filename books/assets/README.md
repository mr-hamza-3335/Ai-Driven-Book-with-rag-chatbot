This folder contains images and diagrams for the Physical AI & Humanoid Robotics textbook.

```
assets/

│   ├── humanoid-anatomy.svg
│   └── ai-stack.svg

│   ├── ros2-architecture.svg
│   └── topic-communication.svg
├── chapter-03/          # Gazebo assets
│   ├── gazebo-ui.png
│   └── physics-properties.svg
├── chapter-04/          # NVIDIA Isaac assets
│   ├── isaac-gym-parallel.svg
│   └── sim-to-real.svg
├── chapter-05/          # VLA assets
│   ├── vla-architecture.svg
│   └── action-tokenization.svg
└── chapter-06/          # Capstone assets
    ├── system-architecture.svg
    └── state-machine.svg
```

## Image Guidelines

- Use SVG for diagrams (scalable)
- Use PNG for screenshots (max 1920x1080)
- Use descriptive filenames
- Include alt text in markdown references
